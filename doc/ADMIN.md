Other music folders can be added via the environment file  `__INSTALL_DIR__/.env`.
```
GONIC_MUSIC_PATH=__DATA_DIR__/music,/path/to/albums,/path/to/compilations # comma separated
```
```
GONIC_MUSIC_PATH=__DATA_DIR__/music,my albums->/path/to/albums,my compilations->/path/to/compilations # comma separated
```
